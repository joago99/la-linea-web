Filtros semánticos aplicables

Clasificación arancelaria

Las bombas de hormigón completas se clasifican bajo el código 8413.40.00. Los componentes y repuestos utilizan la subpartida 8413.91.00. Dado que esta última también incluye partes para otras categorías de bombas, el análisis requiere filtros semánticos adicionales.

Filtrado de microdatos

Se requiere un proceso de tres etapas: 1) seleccionar la partida 8413.91.00 y códigos de importación real; 2) filtrar por marca declarada incluyendo variantes ortográficas; 3) filtrar por palabras clave en la descripción de mercancía.

Fuentes oficiales y triangulación

Los datos de aduana están anonimizados. Para identificar actores reales se recomienda usar Trade Map (directorio de empresas), INAPI (marcas y representantes) y el Directorio Exportador de ProChile para el segmento de proveedores internacionales.

Nota técnica

Los datos presentados en este proyecto se construyen a partir de fuentes públicas chilenas. Una build automatizada de tablas Aduana/ODEPA requiere cron mensual y filtros de texto sobre marcas/descripciones, lo cual está fuera del alcance actual de dashboard.
